云服务器Vshell远控+钓⻥⽊⻢制作

发表于2025-10-09|更新于2026-02-14|网络安全渗透学习
|总字数:608|阅读时长:2分钟|浏览量:

1.申请服务器

建立云服务器通过在阿里云申请一台linux Ubuntu的服务器，以Ubuntu 24.04作为服务器操作系统，

2.下载FinalShall

使用FinalShall来管理云服务器：需要先建立连接配置公网ip用户名和密码

3.部署Vshell

将Vshell部署到云服务器上，首先将Vshell文件上传到云服务器上使用unzip命令对压缩包进行解压，并修改配置文件（主要修改web_username，web_password），防止暴力破解，导致服务器陷入危险

4.防火墙管理

配置防火墙：开放8082和8084端口用来自己进入

5.设置服务端的权限

为服务端的主程序添加可执行的权限：首先进入配置文件的文件夹cd v_linux_amd64/ 设立权限：chmod 777 v_linux_amd64

6.服务器启动

使用命令来启动服务器./v_linux_amd64

7.进入攻击服务器

通过访问公网ip:8082，输入自己配置文件中的密码来进入Vshell的管理界面
设立新的监听路线：配置监听地址使用8084端口进入，使用外网地址连接自己的服务器

8.生成木马
在客户端生成界面选择相应的配置来生成相应的木马文件（下载木马文件时最好关闭杀毒软件）
在客户端管理界面可以对被攻击机进行控制目前可用的功能查看并更改文件，偷窥电脑屏幕，控制电脑（有概率掉线）

对生成的木马进行外壳包装

1.配置rust

对rust进行配置官网上下载后依次操作

2.配置MSVC工具链

MSVC工具链的配置：经过多次尝试后发现不需要下载Visual Studio Installe可以通过命令进行安装rustup install nightly-2024-06-26-x86_64-pc-windows-msvc和rustup default nightly-2024-06-26-x86_64-pc-windows-msvc

3.激活MSVC工具链
需要对MSVC工具链进行激活可以用rustup default nightly-2024-06-26-x86_64-pc-windows-msvc将其设置为默认工具链或者使用rustup run nightly-2024-06-26-x86_64-pc-windows-msvc cargo build命令激活

4.对木马进行包装

进入程序之后可以对下载的木马文件进行外壳包装
包装过程还未进行过多了解
注意：经过测试好像只有.bin的文件可以进行外壳包装

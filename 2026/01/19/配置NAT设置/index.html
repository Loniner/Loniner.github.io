配置NAT设置

basic NAT 设置

首先为各个设备配置ip

将路由器<1>定位为NAT路由器

在路由器<1>上配置默认路由，将所有未知目的地的流量发送到192.168.56.2

ip route-static 0.0.0.0 192.168.56.2

允许源地址为10.0.0.0/24的流量

acl number 2000

rule 0 permit source 10.0.0.0 0.0.0.255

设置地址池<1>

nat address-group 1

地址池从192.168.56.11到192.168.56.20

address 192.168.56.11 192.168.56.20

进入连接外网的接口

int g0/1

将设置的配置启用，将ACL 2000匹配的流量使用地址组1进行转换，no-pat：关键参数，表示不进行端口转换

nat outbound 2000 address-group 1 no-pat

PS:

192.168.56.0/24是私有地址段

真正的NAT应该使用公网IP地址

但在实验环境中可以使用，只要能连通测试

NAPT 设置

NAPT 与 basic NAT 区别

1.NAPT在设置地址池时可以设置1个（由于其可以在一个ip下划分端口）
而basic NAT不能只设置一个如果只设置一个，在这个网络下就只有一个pc可以上网，其他pc不能上网

address 192.168.56.11 192.168.56.11

2.二者最后在端口上的启用命令不同

将设置的配置启用，将ACL 2000匹配的流量使用地址组1进行转换，表示进行端口转换，此处不用 no-pat 

nat outbound 2000 address-group 1

Easy IP 设置

NAT设备直接使用出接口的IP地址作为转接后的源地址，其工作原理与普通NAPT相同，时NAPT的一种特例，适用于拨号接入Internet或动态获取IP地址的场合

Easy IP 无需创建IP 池 只需配置ACL和地址转换即可

acl number 2000

	rule 0 permit source 10.0.0.0 0.0.0.255

int g0/1

undo nat outbound 2000
